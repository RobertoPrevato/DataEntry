# Files for documentation
This folder contains files used to produce documentation.

## Schemas
The .wsd files contained in this folder are used to generate UML graphs using PlantUML server and VS Code jebbs extension.

Useful links:
* [http://plantuml.com](http://plantuml.com)
* [https://github.com/qjebbs/vscode-plantuml](https://github.com/qjebbs/vscode-plantuml)
